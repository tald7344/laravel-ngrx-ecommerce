1- Open the site (https://beyondco.de/docs/laravel-websockets/getting-started/installation) 
   and follow the steps to install 'laravel-websockets' in your backend section


2- Go To file 'config/app.php' and uncomment the line :
	App\Providers\BroadcastServiceProvider::class,


3- Open '.env' file and insert your custom sockets key as following :
	PUSHER_APP_ID=1234567
	PUSHER_APP_KEY=as6df655a4sd6f156
	PUSHER_APP_SECRET=asdf322ad6s6846320ad
	PUSHER_APP_CLUSTER=mt1


4- Install Laravel-echo in frontend section using :
	npm install --save-dev laravel-echo pusher-js


5- Add Pusher To Frontend Section 
   By visit the pusher documentation (https://pusher.com/docs/channels/getting_started/javascript/?ref=docs-index)


6- store (https://js.pusher.com/7.0.3/pusher.min.js) file in your assets folder and add it to 'angular.json' file 
	"scripts": [
		...
              	"src/assets/js/pusher.min.js"
        ]


7- Create Channel In Backend : php artisan make:event ChatEvent

8- create our components in frontend section using :
	ng g m admin/pages/chat
	ng g c admin/pages/chat/components/chat
	ng g s admin/pages/chat/services/chat


9- Create our listener in frontend


10- Run laravel server and websockets server using :
	php artisan serve
	php artisan websockets:serve


11- Send Event from tinker for test :
	php artisan tinker
	event(new App\Events\ChatEvent('Hello'))
